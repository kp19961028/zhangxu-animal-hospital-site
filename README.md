# 浙江张旭动物医院网站

这是从 Stitch 导出的设计整理出的静态多页面站点。

## 页面

- `index.html`：首页
- `about.html`：医院介绍
- `services.html`：科室与诊疗服务
- `team.html`：专家团队
- `guide.html`：就诊指南
- `contact.html`：联系我们
- `doctor-zhangxu.html`：张旭医生详情页

## 本地预览

在这个目录运行：

```powershell
node server.mjs 4173
```

然后访问：

```text
http://localhost:4173
```

`assets/screens/` 中保留了 Stitch 原始截图，便于对照设计稿。
